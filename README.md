# antonki.tutor

PWA-прототип кабинета репетитора с тремя ролями: учитель, ученик и родитель.

## Запуск

```bash
python3 -m http.server 4173
```

Откройте `http://localhost:4173`.

## Что уже есть

- Ученик: курсы, модули, уроки, видео, материалы, задания, тесты, статусы прохождения.
- Учитель: сводка по ученикам, журнал результатов, конструктор курсов, модулей, уроков и тестовых вопросов.
- Родитель: прогресс, оценки, динамика и список ближайших задач.
- PWA: `manifest.webmanifest`, service worker, локальное хранение демо-данных.

Данные сохраняются в `localStorage`. Кнопка сброса возвращает демо-состояние.
